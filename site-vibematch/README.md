# Site explicativo do VibeMatch

Este site está em pasta separada do projeto principal do bot.

## Estrutura
- `index.html`
- `styles.css`
- `script.js`

## Objetivo
Explicar para novos membros do servidor:
- propósito do VibeMatch,
- fluxo de uso,
- todas as funções/comandos existentes,
- regras de convivência e segurança.

## Como abrir
Abra `site-vibematch/index.html` no navegador.

## Publicação (opcional)
Você pode publicar esta pasta em qualquer hospedagem estática (GitHub Pages, Netlify, Vercel static).
